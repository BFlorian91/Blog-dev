Ici, je vais vous expliquez comment faire un system de routes simple avec flutter.

A la racine de votre projet dans le Widget nommé par default ***MyApp***, déclarez vos routes comme ceci:

```js
  return MaterialApp(
    home: HomePage(),
    routes: {
      '/home-page': (context) => HomePage(),
      '/profile': (context) => Profile(),
    },
  )
```

Ensuite il suffit de faire un button dans votre HomePage par exemple et mettre dans le *OnPressed*:
```js
  Navigator.pushNamed(context, '/profile');
```

et vous serez directement redirigé vers la dite page.
